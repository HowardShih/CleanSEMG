# STCNet — Attribution

The code in this directory is adapted from:

> Yang, Jaemo, Doheun Cha, Dong-Gyu Lee, and Sangtae Ahn.
> "STCNet: Spatio-Temporal Cross Network with subject-aware contrastive
> learning for hand gesture recognition in surface EMG."
> *Computers in Biology and Medicine*, vol. 185, p. 109525, 2025.
> DOI: [10.1016/j.compbiomed.2024.109525](https://doi.org/10.1016/j.compbiomed.2024.109525)
> Original code: [https://github.com/jaemoyang/STCNet](https://github.com/jaemoyang/STCNet)

## Modifications for CLEANSEMG

- Input pipeline adapted to accept pre-denoised `.mat` and `.pkl` files
  generated by `prepare_neural.py` / `prepare_classical.py`.
- Evaluation scripts extended with a `--denoiser` flag to allow
  systematic comparison across all 8 CLEANSEMG denoising methods.
- Subject-level result aggregation added for macro-F1 reporting across subjects.
- All original model architecture, loss functions, and training
  hyperparameters are unchanged from the published version.
